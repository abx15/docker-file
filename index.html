Docker File System

Understand Docker Architecture & File Structure

Docker Images

Docker images are read-only templates used to create containers. They contain application code, libraries, and dependencies.

Docker Containers

Containers are running instances of Docker images. They are lightweight, fast, and isolated environments.

Volumes

Volumes help persist data generated by containers and allow sharing data between containers safely.

Bind Mounts

Bind mounts map local directories to containers, commonly used during development for live updates.
